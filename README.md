## ğŸ¯ About the Game

**Simon Says** is a classic electronic memory game that challenges players to repeat increasingly longer sequences of colored lights and sounds. This web-based version recreates the nostalgic 1978 Milton Bradley game with a modern, sleek interface.

Test your memory, concentration, and pattern recognition skills as you try to beat your high score! Each level adds a new color to the sequence, making it progressively more challenging.

## ğŸ•¹ï¸ How to Play

### Starting the Game
1. Open the game in your web browser
2. **Press any key** on your keyboard to start
3. Watch the colored button flash
4. Click the button that flashed

### Continuing
5. The game will repeat the sequence and add one more color
6. Repeat the entire sequence by clicking the buttons in order
7. Each successful round increases your level and adds another color to remember

### Game Over
8. If you click the wrong button, the game ends
9. Your final score and high score will be displayed
10. Press any key to play again and try to beat your high score!
